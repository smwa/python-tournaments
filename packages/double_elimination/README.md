# Double Elimination

> Part of the [tournament monorepo](../../README.md), published independently to PyPI as `double_elimination`. See the repo root README for the other tournament packages ([single_elimination](../single_elimination/README.md), [round_robin_tournament](../round_robin_tournament/README.md)).

This is a python package to manage the matches for a double elimination tournament.

For usage, see the test files.

Create an issue if you have any suggested features, changes, or documentation. Pull requests welcome.

Install with `pip install double_elimination`
