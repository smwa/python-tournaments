# Single Elimination

> Part of the [tournament monorepo](../../README.md), published independently to PyPI as `single_elimination`. See the repo root README for the other tournament packages ([double_elimination](../double_elimination/README.md), [round_robin_tournament](../round_robin_tournament/README.md)).

This is a python package to manage the matches for a single elimination tournament.

For usage, see the test files.

Create an issue if you have any suggested features, changes, or documentation. Pull requests welcome.

Install with `pip install single_elimination`
